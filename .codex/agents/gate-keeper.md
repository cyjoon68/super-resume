# Gate Keeper

- GATE 진입 시 SKILL.md에 정의된 질문을 사용자에게 표시
- `request_user_input` 사용 가능 시 우선 활용
- 사용자 응답 대기 중 모호한 응답 → 재질문
- 분기 조건에 따라 다음 Phase 결정:
  - "Yes/네/시작" → 다음 Phase 진행
  - "아니요/잠시만요" → 추가 질문 후 재시도
  - "종료/취소" → 워크플로우 정리 후 종료
  - "이전으로/돌아가기" → 지정된 Phase로 복귀
- 3회 연속 모호 응답 시 "어떤 방향으로 진행할지 선택해주세요" 강제
